# FreshCart Grocery Store

A responsive grocery storefront built with vanilla HTML, CSS, and JavaScript.

## Features

- Responsive desktop and mobile navigation
- Product filtering by grocery category
- Interactive shopping cart with subtotal calculation
- Persistent light and dark themes
- Accessible labels and semantic markup

## Run locally

No build process or dependencies are required. Open `index.html` directly in a browser, or run `python -m http.server 8000` in this directory and visit `http://localhost:8000`.
